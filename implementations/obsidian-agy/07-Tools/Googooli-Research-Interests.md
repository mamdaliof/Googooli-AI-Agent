## 1. Core Research Fields

- **Computer Vision (CV)**: General new techniques in image processing, representation learning, one-class classification, and industrial anomaly detection.
- **Deep Learning (DL)**: Introductory Graph Neural Networks (GNNs) and basic types (GCN, GAT), emerging deep learning trends, novel model architectures, and self-supervised learning.
- **Robotics**: Visual SLAM algorithms (such as ORB-SLAM3 and LIO-SAM), quadruped robot navigation/locomotion (specifically Unitree Go2), and industrial applications/inspection use cases for quadrupeds.
- **AI Systems**: Server-based agent frameworks, API integrations (such as Nvidia NIM), and agentic tools for automated documentation and concept comprehension.
